Jeu Snake Graphique :
	* ce jeu implement plusieurs Libraries java comme swing,awt.event,util.random,,
comment jouer ?
		Le principe de Jeu est de manger le maximum des fruits pour un maximum du score.
		Le serpent  se deplace initialement vers la droit , pour le controle on utilise les clès suivants :
			Clès	|	directions
			↑	|	VERS LE HAUT
			↓	|	VERS LE BAS
			→	|	VERS LA DROITE
			←	|	VERS LA GAUCHE
			R	|	RESTART LE JEUX(aprés perdre une partie)
